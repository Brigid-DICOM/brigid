## DICOM 分享功能

Brigid 內建強大的 DICOM 分享機制，確保醫療影像在跨團隊協作時，既能保持流暢的資訊交換，又能兼顧資料的安全性

- 自定義識別：可設定「分享名稱」與「描述」，方便追蹤與管理
- 權限控管：
  - 操作權限：支持 `READ` (檢視/下載) 與 `EDIT` (編輯 Tag) 權限
  - 對象管理：可精確指定`特定使用者`或開放`所有人`存取
- 安全機制：
  - 密碼保護：強制要求輸入正確密碼才能存取
  - 有效期限：可設定`有效期限`，連結過期後自動失效
