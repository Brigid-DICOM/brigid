## DICOM Sharing Features

Brigid provides a robust DICOM sharing mechanism to ensure smooth information exchange while maintaining data security during cross-team collaboration.

- Custom Identification: Set "Share Name" and "Description" to track and manage easily.
- Permission Control:
    - Operation Permissions: Supports `READ` (view/download) and `EDIT` (edit tag) permissions.
    - Object Management: Specify `Specific Users` or `Everyone` access.
- Security Mechanisms:
    - Password Protection: Requires entering the correct password to access.
    - Expiration Date: Set `Expiration Date`, the link will automatically expire after the expiration date.
